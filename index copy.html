SVG - Scalable Vector Graphics is a vector-based format for web developers. This type of format has become so popular due to the fact that its images can be displayed at high resolutions without loss of quality, since SVG is a vector format.
Images can be downloaded by paid subscription.

SVG is lightweight compared to other formats. This format has incredible features. Flags, symbols, interface elements. And this is just a small list of how to use it.

Next to this text are pictures of the modern web-format .svg image, they slow down page loading in the browser less.

The impetus for the development of this language markup was first given by the World Wide Web Consortium, which was better known in 1999 as the W3C. W3C introduced the concept of SVG - language markup for creating two-dimensional graphical interfaces and images.

The biggest plus of its use is that it is a vector format, that is, it can be used on any displays - whatever their resolution, it will be displayed the same everywhere.
